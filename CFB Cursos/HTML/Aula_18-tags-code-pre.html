O segmento de código a seguir representa o corpo básico de um programa em C.

        
#include ‹stdio.h›
#include ‹stdlib.h›
#include ‹locale.h›

int() main()
{
    setlocale(LC_ALL,"Portuguese");
    printf("Código básico em C");
    system("pause");

    return 0;
}

Os cabeçalhos utilizados podem variar de acordo com os recursos que deseja utilizar no programa, então, stdlib e locale podem ser omitidos como parte de código básico, assim como a função setlocale.
